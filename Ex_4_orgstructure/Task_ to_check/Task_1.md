# Task 1

## Условие

Найти всех сотрудников, подчиняющихся Ивану Иванову (`EmployeeID = 1`), включая их подчиненных, подчиненных подчиненных, а также самого Ивана Иванова.

Для каждого сотрудника нужно вывести:

- `EmployeeID`;
- имя сотрудника;
- `ManagerID`;
- название отдела;
- название роли;
- названия проектов, к которым относится сотрудник;
- названия задач, назначенных сотруднику.

Если у сотрудника нет назначенных проектов или задач, необходимо отобразить `NULL`.

### Требования

- нужно рекурсивно извлечь всех подчиненных Ивана Иванова;
- решение должно быть представлено одним SQL-запросом;
- в запросе должно использоваться ключевое слово `RECURSIVE`;
- результаты нужно отсортировать по имени сотрудника.

## Решение

```sql
WITH RECURSIVE Employe AS (
    SELECT
        EmployeeID,
        Name,
        ManagerID,
        DepartmentID,
        RoleID
    FROM Employees
    WHERE EmployeeID = 1

    UNION ALL

    SELECT
        e.EmployeeID,
        e.Name,
        e.ManagerID,
        e.DepartmentID,
        e.RoleID
    FROM Employees e
    INNER JOIN Employe eh ON e.ManagerID = eh.EmployeeID
)
SELECT
    eh.EmployeeID,
    eh.Name AS EmployeeName,
    eh.ManagerID,
    d.DepartmentName,
    r.RoleName,
    (
        SELECT GROUP_CONCAT(DISTINCT p.ProjectName ORDER BY p.ProjectName SEPARATOR ', ')
        FROM Tasks t
        JOIN Projects p ON t.ProjectID = p.ProjectID
        WHERE t.AssignedTo = eh.EmployeeID
    ) AS ProjectNames,
    (
        SELECT GROUP_CONCAT(DISTINCT t.TaskName ORDER BY t.TaskName SEPARATOR ', ')
        FROM Tasks t
        WHERE t.AssignedTo = eh.EmployeeID
    ) AS TaskNames
FROM Employe eh
LEFT JOIN Departments d ON eh.DepartmentID = d.DepartmentID
LEFT JOIN Roles r ON eh.RoleID = r.RoleID
GROUP BY eh.EmployeeID, eh.Name, eh.ManagerID, d.DepartmentName, r.RoleName
ORDER BY eh.Name;
```
